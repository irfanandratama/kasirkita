## About KasirQ

KasirQ - Sistem kasir berbasis web untuk semua jenis bisnis. Sistem ini memiliki 3 jenis user, diantaranya yaitu :
- Admin = Sebagai super user yang memiliki hak akses penuh terhadap user
    
- Management = Sebagai pemilik bisnis atau usaha, user jenis ini hanya bisa didapatkan dengan cara register. Sebagai pemilik usaha maka user jenis ini dapat melihat laporan penjualan pada semua outlet yang dimilikinya, jumlah pendapatan, dan juga dapat melihat grafik jumlah transaksi hingga produk best seller
    
- Kasir = User jenis ini hanya dapat ditambahkan oleh user jenis management. user jenis ini dapat melakukan transaksi dan melihat stok barang
